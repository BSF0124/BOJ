# 💡 문제 접근

- 홀수는 최대 K개까지 제거할 수 있다 → **구간 내 홀수 개수를 세면서 투 포인터 확장**
- 짝수 개수는 **총 길이 - 홀수 개수**로 계산 가능
- 따라서 구간을 `[st, en]`으로 설정했을 때,
    
    해당 구간 내의 짝수 개수는 `en - st + 1 - 홀수 개수`
    

---

# 🧩 풀이 방식

1. **`st`는 왼쪽 포인터**, 0부터 시작
2. **`en`은 오른쪽 포인터**, 가능한 한 멀리 확장
3. 구간 `[st ~ en]` 내의 **홀수 개수 `cnt`가 K 이하일 때만** 확장 가능
4. 매 반복마다 `현재 구간의 짝수 개수 = (en - st + 1 - cnt)`를 계산해 `mx`값 갱신
5. 이후 `st`를 한 칸 이동시키면서, 필요시 `cnt`도 감소
